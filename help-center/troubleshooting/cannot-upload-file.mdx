---
title: Cannot Upload File
description: Resolve file upload problems on Umashankar Sahu.
---

## Symptoms

- Upload button shows error
- File does not appear in project

## Possible Causes

- File size exceeds 5MB limit
- Unsupported file format

## Quick Fix

<Steps>
  <Step title="Check file size">
    ````
    Ensure your file is under 5MB.
    ````
  </Step>
  <Step title="Supported formats">
    ````
    Allowed: .md, .pdf, .txt
    ````
  </Step>
  <Step title="Try a different browser">
    ````
    Sometimes browser extensions block uploads. Try Chrome or Firefox.
    ````
  </Step>
</Steps>
