## Overview

Follow these steps to set up your first project.

<Steps>
  <Step title="Open Dashboard">
    ````
    Visit https://umashankarsahu.com and log in.
    ````
  </Step>
  <Step title="Start New Project">
    ````
    Click "Create New Project" on the top right.
    Enter a project name and pick a template.
    ````
  </Step>
  <Step title="Add Documentation Pages">
    ````
    Use the built-in editor to add your first page.
    Click "Save" to publish.
    ````
  </Step>
</Steps>
